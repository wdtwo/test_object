自定义 Ajax

使用 ajax 选项，来替换ajax调用的方法。应该与 jQuery ajax 方法具有相同的API。

ID 条目名称 条目价格

方法

方法名 说明 demo

getOptions

使用 ajax 选项，来替换ajax调用的方法。应该与 jQuery ajax 方法具有相同的API。 $table.bootstrapTable('getOptions');

getSelections

返回所选的行，当没有选择任何行的时候返回一个空数组 $table.bootstrapTable('getSelections');

getAllSelections

返回所有选择的行，包括搜索过滤前的，当没有选择任何行的时候返回一个空数组 $table.bootstrapTable('getAllSelections');

getData

获取当前加载的数据 $table.bootstrapTable('getData');

getRowByUniqueId

根据 uniqueId 获取行数据 $table.bootstrapTable('getRowByUniqueId', 1);

load

加载数据到表格中，旧数据会被替换 $table.bootstrapTable('load', data);

append

添加数据到表格在现有数据之后 $table.bootstrapTable('append', data);

prepend

插入数据到表格在现有数据之前 $table.bootstrapTable('prepend', data);

remove

从表格中删除数据，包括两个参数： field: 需要删除的行的 field 名称，values: 需要删除的行的值，类型为数组。 $table.bootstrapTable('remove', {field: 'id', values: ids});

removeAll

删除表格所有数据 $table.bootstrapTable('removeAll');

removeByUniqueId

根据 uniqueId 删除指定的行 $table.bootstrapTable('removeByUniqueId', 1);

insertRow

插入新行，参数包括： index: 要插入的行的 index， row: 行的数据，Object 对象 $table.bootstrapTable('insertRow', {index: 1, row: row});

updateRow

更新指定的行，参数包括： index: 要更新的行的 index， row: 行的数据，Object 对象 $table.bootstrapTable('updateRow', {index: 1, row: row});

updateByUniqueId

更新指定的行，参数包括： id: 行 id，id 应该为 uniqueid， row: 新行的数据，Object 对象 $table.bootstrapTable('updateByUniqueId', {id: 3, row: row});

showRow/hideRow

显示/ 隐藏指定行 $table.bootstrapTable('showRow', {index:1}); $table.bootstrapTable('hideRow', {index:1});

mergeCells

将某些单元格合并到一个单元格，选项包含以下属性：index: 行索引，field: 字段名称，rowspan: 要合并的rowspan数量，colspan: 要合并的colspan数量 $table.bootstrapTable('mergeCells', {index: 1, field: 'name', colspan: 2, rowspan: 3});

checkAll/uncheckAll

选中/取消选中当前页面所有行 $table.bootstrapTable('checkAll'); $table.bootstrapTable('uncheckAll');

check/uncheck

选中/取消选指定行，行索引从0开始 $table.bootstrapTable('check', 1); $table.bootstrapTable('uncheck', 1);

checkBy/uncheckBy

从数组值中选中/取消选行 $table.bootstrapTable('checkBy', {field:'id', values:[1, 2, 3]}); $table.bootstrapTable('uncheckBy', {field:'id', values:[1, 2, 3]});

refresh

刷新远程服务器数据，可以设置 {silent: true} 以静默方式刷新数据，并设置 {url: newUrl} 更改URL。 要提供特定于此请求的查询参数，请设置 {query: {foo: 'bar'}}。

resetView

重置引导表视图，如例如重置表高度 $table.bootstrapTable('resetView');

destroy

销毁表格 $table.bootstrapTable('destroy');

showLoading/hideLoading

显示/隐藏加载状态 $table.bootstrapTable('showLoading'); $table.bootstrapTable('hideLoading');

showColumn/hideCoulumn

显示/隐藏指定列 $table.bootstrapTable('showColumn', 'name'); $table.bootstrapTable('hideCoulumn', 'name');

scrollTo

滚动到指定位置，单位为 px，设置 'bottom' 表示跳到最后 $table.bootstrapTable('scrollTo', 0); $table.bootstrapTable('scrollTo', 'bottom');

selectPage/prevPage/nextPage

转到指定页/上一页/下一页 $table.bootstrapTable('selectPage', 1); $table.bootstrapTable('prevPage'); $table.bootstrapTable('nextPage');

togglePagination

切换分页选项 $table.bootstrapTable('togglePagination');

toggleView

切换 card/table 视图 $table.bootstrapTable('toggleView');

refreshOptions

刷新选项 $table.bootstrapTable('refreshOptions');

resetSearch

设置搜索文本 $table.bootstrapTable('resetSearch', '1');

expandRow/collapseRow

如果详细视图选项设置为 True，可展开/收起索引为 index 的行 $table.bootstrapTable("expandRow", 1); $table.bootstrapTable("collapseRow", 1);

filterBy

过滤表格数据（只能用于 client 端）， 你可以通过过滤 {age: 10} 来显示 age 等于 10 的数据。 您也可以使用数组来过滤，如：{age: 10, hairColor: ["blue", "red", "green"]} 找到 age 等于 10 的数据，hairColor 可以是 blue，red 或 green。 在本示例中，我们使用：$table.bootstrapTable('filterBy', {id: [1, 2, 3]});
